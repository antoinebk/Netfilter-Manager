Netfilter-Manager by antoinebk

Netfilter Manager is a small application allowing Linux system admins to manage the netfilter rules of all their servers.

Netfilter Manager is used through an intuitive Command Line Interface. It can currently push rules by SSH and execute them on the remote host.

Features :

– Manage many Linux hosts – Create per-host iptables rules – Organize your iptables rules – Flexible template system allowing you to quickly create sets of rules for your hosts – Push the rules via SSH onto the remote host and apply them remotly Bonus for those not familiar with iptables : – Add rules using Cisco ACL syntax (only IP rules for now)
